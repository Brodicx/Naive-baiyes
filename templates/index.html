Gaussian Naive Bayes - Diabetes

Landing sederhana. Aplikasi utama berjalan di app.py (Streamlit). Dataset: diabetes.csv.

Langkah Menjalankan

  • Pastikan Python 3 terpasang.
  • Install deps: pip install streamlit scikit-learn pandas seaborn matplotlib
  • Jalankan: streamlit run app.py
  • Buka URL yang muncul (default http://localhost:8501).

Fitur Aplikasi

  • Training cepat GaussianNB pada diabetes.csv (scaling wajib).
  • Akurasi & Confusion Matrix (heatmap).
  • Analisis predict_proba untuk sampel uji.
  • Simulasi prediksi data baru (input manual).

Koneksi dengan Notebook

Catatan: app.py berdiri sendiri dan memuat ulang pipeline yang sama dengan notebook diabetes_nb.ipynb. Tidak ada pemanggilan langsung ke notebook; semua preprocessing/training diulang di app.py menggunakan diabetes.csv.
